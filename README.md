# xls-viewer


## Overview

CLI-based Excel viewer


## How to use

- `$ node app [options] <filepath>`
  - `[options]`
    - `--sheet=sheetname`
      - specify single sheet by name
      - default: all sheets
    - `--row_max_width=num`
      - specify maximum width for each rows
      - default: 20
    - `--border=num`
      - specify 0 if border should not be displayed
      - default: 1
    - `--value=num`
      - specify 0 if formula should be displayed
      - default: 1
